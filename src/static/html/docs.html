Get SQL Table contents

This uses /post method and retrieves the table data from mysql server based on the database and table name provided. Note: ! - means optional

Request Object

                    {
                        "request":{
                            "d_name":"Database name",
                            "t_name":"Table name",
                            "fields"!: ["fields array"]
                        }
                    }

Response Object

                    {
                        "response":{
                            "code":100-501,
                            "status":"Status text",
                            "result":{{
                                "msg":"[{response}] | Response msg"
                            }}
                        }
                    }

Filter SQL Table contents

This is /post method which returns the data based on the filtering condition. Note: ! - means optional

Request Object

                    {
                        "request":{
                            "d_name":"Database name",
                            "t_name":"Table name",
                            "fields":["Array of fields name"]
                            "filter": [
                                    "Array of filter",
                                    ["col_name", "relations", "value", "is_not"],
                                    "relations" => "-eq, -lte, -gte-, -ne, -like, -lt, -gt"
                                    "is_not" => "whether condition is enclosed within NOT()"
                                    "logical-relation" !optional,
                                    "logical-relation" => "and, or"
                                    ["col_name", "relations", "value", "is_not"] !optional
                                    ...
                                ]
                        }
                    }

Response Object

                    {
                        "response":{
                            "code":100-501,
                            "status":"Status text",
                            "result":{{
                                "msg":"[{response}] | Response msg"
                            }}
                        }
                    }

Create SQL Table

This is a /post method which allows user to create a sql table based on the specifications provided by the user. Note: ! - means optional

Request Object

                    {
                        "request":{
                            "d_name":"Database name",
                            "t_name":"Table name",
                            "fields":[
                                        {
                                            "name":"name of the column",
                                            "type":"data type of the column",
                                            "length" !:"length of the column",
                                            "is_primary" !:"boolean whether is primary key or not",
                                            "is_unique" !:"boolean whether column is unique or not",
                                            "not_null" !:"boolean whether column is not null or not",
                                            "auto_increment" !: "boolean whether is auto increment or not"
                                        }
                                        ,...
                                    ]
                            }
                        
                    }

Response Object

                        "response":{
                            "code":100-501,
                            "status":"Status text",
                            "result":{{
                                "msg":"[{response}] | Response msg"
                            }}
                        }
                    }
